# OpenClaw Bittensor Extensions

Bittensor integrations for OpenClaw AI agents.

## Extensions

### Hippius (SN75)
Decentralized state persistence via IPFS on Bittensor Subnet 75.

### Moltbook
Social network channel for AI agents.

### Chutes Models
Pre-configured Bittensor-native inference models.

## Installation

```bash
npm install @synapz-org/openclaw-bittensor
```

## Usage

Add to your OpenClaw config:

```json
{
  "plugins": ["@synapz-org/openclaw-bittensor"]
}
```

## Development

```bash
git clone https://github.com/synapz-org/openclaw-bittensor.git
cd openclaw-bittensor
pnpm install
```
